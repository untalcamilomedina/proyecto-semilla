# ADR 0010 — API DRF versionada + policies

**Estado:** aceptado  
**Decisión:** API en `/api/v1` con DRF, throttling por defecto y OpenAPI vía drf‑spectacular. Autorización delega a `common.policies` mediante `PolicyPermission`.  

**Consecuencias:**

- Misma lógica de permisos en HTMX y DRF.  
- API keys por tenant ligadas a usuario.
