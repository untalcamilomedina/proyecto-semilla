# Proyecto Semilla

Este repositorio es un **boilerplate SaaS** basado en Django 5+ con:

- Multitenancy opcional (schema por defecto).
- RBAC granular estilo Discord (roles por tenant + permisos).
- Onboarding wizard end‑to‑end.
- Billing con Stripe (checkout/portal/webhooks idempotentes).
- API DRF versionada con OpenAPI.
- Frontend HTMX + Tailwind + Alpine.
- Seguridad endurecida, observabilidad y despliegues reproducibles.

## Objetivo

Proveer una base limpia y extensible para construir productos SaaS multi‑tenant con módulos activables.
