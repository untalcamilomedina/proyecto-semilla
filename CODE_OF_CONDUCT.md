# Code of Conduct

Este proyecto sigue el Contributor Covenant.

## Nuestra promesa

Nos comprometemos a crear un entorno abierto y respetuoso para todos.

## Estándares

Comportamientos aceptables incluyen:

- Ser respetuoso y empático.
- Aceptar críticas constructivas.
- Priorizar el bien común del proyecto.

Comportamientos inaceptables incluyen:

- Acoso o discriminación.
- Lenguaje ofensivo.
- Conductas hostiles o intimidatorias.

## Aplicación

Los mantenedores pueden remover contribuciones que violen este código.
